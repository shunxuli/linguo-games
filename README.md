# 果果乐园 (linguo-games)

儿童教育游戏平台，包含数独、数学运算和拼图三款小游戏。

## 技术栈

- Vue 3 (Composition API)
- TypeScript
- Pinia (状态管理)
- Vite (构建工具)
- Vitest (单元测试)

## 开发

```bash
npm install
npm run dev      # 启动开发服务器
npm run lint     # 代码检查
npm run test     # 运行测试
npm run build    # 构建生产版本
```

## 部署

构建产物为纯静态文件，可部署到任意静态文件服务器。
